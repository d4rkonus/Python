```python
#!/usr/bin/python3

students = []
scores = []

for _ in range(int(input())):
    name = input()
    score = float(input())
    students.extend([[name, score]])

for student in students:
    scores.append(student[1])

unique_scores = sorted(set(scores))
second_lowest_score = unique_scores[1]

names = []
for student in students:
    if student[1] == second_lowest_score:
        names.append(student[0])

names.sort()
for name in names:
    print(name)
```

Empezamos creando 2 listas que aunque parezcan vacías, contendrán los datos que se nos proporcionan. 

El primer bucle **for** toma el dato de entrada, lo convierte a número entero y lo añade a un rango de valores.

Posteriormente, Python almacena los nombres de los estudiantes en *name*, y su puntuación en *score*, para después meter ambos datos a la lista **students**. 

El segundo bucle **for** recorre todos los datos de la lista **students**, y añade las puntuaciones *score* a la lista **scores**. 

Con esto se consigue que se reúnan todos los datos  en distintas variables, para después separarlos en listas, según su tipo de dato.

Posteriormente creamos 2 variables que organizan las puntuaciones y seleccionan la puntuación mas baja. 

La variable **unique scores** almacena las puntuaciones de forma ordenada y no permite que se repiten las mismas puntuaciones. 

La variable **second_lowest_score** almacena el segundo elemento del conjunto de  puntuaciones de **unique scores**. 

Posteriormente creamos una lista vacía llamada *names* que es donde guardaremos el nombre de estudiantes que tengan la misma puntuación de **second_lowest_score**.

Después aparece un bucle **for** que recorre la lista de estudiantes y se añade un condicional que incluye la puntuación del estudiante, si ésta es igual a la que se guarda en **second_lowest_score**. 

Y para mostrar los resultados, al final del programa, se ordenan los nombres y se muestran por pantalla.
