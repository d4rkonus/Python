```python
if __name__ == "__main__":
	n = int(input())
	student_marks = {}

	for i in range(n):
		data = input().split()
		# The first element is the name
		name = data[0]
		# list where save the scores
		scores = []
		
		# We take the elements from the position 1 to the end
		for score in data[1:]:
			scores.append(float(score))

		student_marks[name] = scores

	query_name = input() 

	scores = student_marks[query_name]
	total = sum(scores)
	count = len(scores)
	average = total / count

	print(f"{average:.2f}")
```

Primero se declara una variable **n** que recibe la cantidad de estudiantes que se van a procesar. 

Después se crea un diccionario **students_marks** que guardará pares de clave-valor donde la clave es el nombre de estudiante y el valor es la lista con las notas por cada estudiante.

Se ejecuta un bucle **for** que se repite **n** veces, una por cada estudiante. Dentro de este bucle, la variable **data** recibe una línea de entrada y la separa en una lista usando *split()*. 

Se crea un bucle *for* anidado (segundo bucle dentro del primero) que recorre todos los elementos de **data** a partir del segundo, y los añade a la lista **scores** en formato flotante.

Se guarda en el diccionario los pares de clave valor, el nombre de los estudiantes junto con sus notas, y a partir de aquí se crea una variable **query_name** que recibe el nombre del estudiante que se desea consultar.

Se crea una variable **scores** que recibe la lista de notas junto con el nombre de la consulta, y a partir de sumarlas todas las notas de cada estudiante y dividir el resultado de la suma entre la cantidad de las notas, se calcula la media del estudiante consultado en *query_name*.
